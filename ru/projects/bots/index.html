Encoder Bot

@CoderTextBot

Демонстрация функционала

Бот шифровует и расшифровует текст, после чего выдает уникальный ключ 🔑

После отправки любого сообщения, бот отправит Вам зашифрованное сообщение и уникальный ключ, с помощью которого можно это сообщение расшифровать

Список комманд
/start
Запуск бота
/settings
Настройки бота
/cancel
Отмена действий
Режимы
Шифр Цезаря
Этот режим позволяет шифровать текст заменяя символом находящимся на некотором постоянном числе позиций левее или правее него в алфавите
Шифр Виженера
Этот режим позволяет шифровать текст с использованием ключевого слова, которое можно придумать самому, или сгенерировать автоматически

KhNURE Bot

@pripadoc_bot

Демонстрация функционала

Готовые задания по предметам

Бот отправляет готовые задания по предметам

Для того чтобы найти нужное задание по предмету,
просто напишите /start, после чего выберите Курс > Предмет > Задание

Список комманд
/start
Запуск бота
/info
Информация
/donate
Поддержать разработчика

Некоторые задания платные, цена написана в скобках

Пример: 2 курс\ОТПВ\ПЗ 3 отчет (30грн)
